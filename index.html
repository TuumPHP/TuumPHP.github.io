TuumPHP is a basic web application framework.

Yet Another Framework...

TuumPHP is a basic (micro + view) web application framework for normal web sites.

Inspired by clean middleware architecture of StackPHP, simplicity of Slim Framework, and ease of development of Laravel.

This project started as a build-own framework to learn programming by using many other reliable packages as well as the latest PSR-7 http-message. More documents and tests shall be written to complete this project...

Status

Alpha release.

It works as is, but API may change at any time, and not fully tested yet.

Installation

$ git clone https://github.com/TuumPHP/tuum -b 0.1.0
$ cd tuum
$ composer install
$ cd public
$ php -S localhost:8888 index.php

please make the var/ directory writable to the server.

Access http://localhost:8888 in a browser.

Requirements

TuumPHP probably runs with PHP5.5 and above.

packages

TuumPHP uses following packages (if not all)!

  • Aura/Session,
  • Phly/Http,
  • Psr/Http-message,
  • Monolog/Monolog,
  • PHP League/CommonMark,
  • PHP League/Container,
  • PHP League/Flysystem,
  • Filp/Whoops.

the next release will use Zend-Frameworks' Diactoros.
